Data Cleaning procedures

Clean Sample

  • Initial Size: 2449
  • Remove blank email addresses (105 in raw data).
  • period column created with values fall12 and spring13 corresponding to TRUE values of fall12dum and spring13dum respectively.
  • Removed responses from those with more than 10 semesters of college. (33 in raw data)
    • Note that the clean sample data and price beliefs data appear to have different values of semester and freshdum for the same instances of email and period (loose join conditions). No explanation as of yet.
  • Final size: 2311

Price Beliefs

  • Initial Size: 4785
  • Did not remove unfinished(?) (finished == 0) surveys (537 did not finish?)
  • period column created with values fall12 and spring13.
    • startyear and startmonth created from original column startdate
    • period=fall12 set if startyear == 2012 and (startmonth == 11 or startmonth == 12)
    • period=spring13 set if startyear == 2013 and startmonth == 4
  • mu and sigma columns created to enable log-normal price distribution comparisons of respondent’s price distribution for new books. (Uses newexp)
    • mu: 388 observations with NA, NaN, or Inf responses in the final data.
    • sigma: 317 observations with NA, NaN, or Inf responses in the final data.
    • mu and sigma: 577 observations have non-sensical probability responses. (E.g. newlb + newub >= 100 )
    • Normalized normmu also available. (Utilizes newexpratio instead of newexp.)
  • Removed responses from those with more than 10 semesters of college (267 in raw data.)
    • Note that the clean sample data and price beliefs data appear to have different values of semester and freshdum for the same instances of email and period (loose join conditions). No explanation as of yet.
  • Removed expectations of online new price that are < 10% or > 150% of the new bookstore price (12 in raw data.)
  • Final size: 4147
